INSTRUCTIONS FOR 'WORD ATTACK!'
-----

- Select difficult using the 'Easy', 'Medium' and 'Hard' buttons.
- Use the dropdown menus above the challenge box to select your language and vocab list.
- Once all settings have been correctly set, click the 'Start' button to initiate the game. You will have 60 secs to translate as many words as possible, with a timer on each individual word depending on your difficulty setting.
